# Анимации и переходы в вебе

> **Ссылка на GitHub Pages:** https://justwolk.github.io/pzwebsite/css-transitions.html

## О проекте

Учебный многостраничный сайт, посвящённый современным технологиям создания анимаций в вебе: CSS Transitions, CSS Animations, @keyframes и Web Animations API.

Создан в рамках курсовой работы.

- **ФИО:** Цветов Дмитрий Ильдарович

## Тема проекта

Анимации и переходы в вебе (CSS-анимации, Web Animations API и др.)

## CSS-фреймворк

**Primer CSS** — [primer.style/css](https://primer.style/css)

Подключается через CDN: `https://unpkg.com/@primer/css@21.3.3/dist/primer.css`

## Технологии

- HTML5 
- CSS3 
- Primer CSS 
- JavaScript ES6+ 

## Страницы сайта

1. **Главная** (`index.html`) — введение в тему, обзор технологий
2. **CSS-анимации** (`css-animations.html`) — свойство animation, примеры
3. **CSS Transitions** (`css-transitions.html`) — переходы, hover-эффекты
4. **@keyframes** (`keyframes.html`) — ключевые кадры подробно
6. **Web Animations API** (`web-animations-api.html`) — примеры и управление
7. **Контакты** (`contacts.html`) — форма обратной связи

## Реализованные требования

- [x] CSS Grid и Flexbox для сетки
- [x] CSS Variables для темизации
- [x] Тёмная тема
- [x] Адаптивность
- [x] Бургер-меню на мобильных
- [x] Микроанимации (hover, focus, модальные окна)
- [x] Встроенное видео (YouTube iframe, loading="lazy")
- [x] Ленивая загрузка (loading="lazy")
