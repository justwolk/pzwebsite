Web Animations API

Web Animations API (WAAPI) — это нативный браузерный интерфейс, который объединяет возможности CSS-анимаций и JavaScript. Он позволяет создавать, управлять и контролировать анимации прямо из JavaScript-кода.

Зачем нужен WAAPI?

CSS-анимации удобны, но ими сложно управлять из кода: нельзя поставить на паузу, перемотать или динамически изменить параметры. WAAPI решает эту проблему — он даёт программный контроль над анимациями.

Метод element.animate()

Основной метод API — animate(), вызываемый на DOM-элементе:

const box = document.querySelector('.box');

const animation = box.animate([
  { transform: 'translateX(0)', opacity: 1 },
  { transform: 'translateX(300px)', opacity: 0.5 },
  { transform: 'translateX(0)', opacity: 1 }
], {
  duration: 2000,
  iterations: Infinity,
  easing: 'ease-in-out'
});

Управление анимацией

Объект Animation, возвращаемый методом animate(), предоставляет методы управления:

  • animation.play() — запустить
  • animation.pause() — поставить на паузу
  • animation.reverse() — воспроизвести в обратном порядке
  • animation.cancel() — отменить
  • animation.finish() — перейти к концу
  • animation.currentTime — текущее время (для перемотки)
  • animation.playbackRate — скорость воспроизведения

Отличия от CSS-анимаций

Возможность CSS Animations Web Animations API
Описание в CSS Да Нет
Пауза/воспроизведение Через animation-play-state play() / pause()
Перемотка Нет currentTime
Скорость Нет (нужно менять duration) playbackRate
Динамические значения Нет Да
Callback при завершении animationend event animation.finished (Promise)

Поддержка браузерами

Web Animations API поддерживается во всех современных браузерах: Chrome, Firefox, Safari, Edge. Для старых браузеров существует полифилл web-animations-js.

Когда использовать?

WAAPI подходит, когда нужен программный контроль над анимацией, но не хочется подключать тяжёлую стороннюю библиотеку. Это «золотая середина» между CSS и сторонними библиотеками.
